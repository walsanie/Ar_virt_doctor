metrics3.fmeasure(y_true, y_pred)¶

Fmeasure metric. Computes the fmeasure, a metric for multi-label classification and is defind as the weighted harmonic mean of the precision and recall of the test

metrics3.matthews_correlation(y_true, y_pred)¶

Matthews correlation metric. # Aliases

It is only computed as a batch-wise average, not globally. Computes the Matthews correlation coefficient measure for quality of binary classification problems.

metrics3.precision(y_true, y_pred)¶

Precision metric. Only computes a batch-wise average of precision. Computes the precision, a metric for multi-label classification of how many selected items are relevant.

metrics3.recall(y_true, y_pred)¶

Recall metric. Only computes a batch-wise average of recall. Computes the recall, a metric for multi-label classification of how many relevant items are selected.
